	var res = "あいうえお9aabbc123".match('/\\d{2}$/');
	if(res){
		console.log(res[0]); // → 23
		console.log(res['index']); // → 17 (先頭からの位置。先頭は0で、全角は2とカウント)
	}
x:正規表現一致取得文字オフセット（位置）ソースコード説明
※「\」バックスラッシュについて。
バックスラッシュは /\\\\/ とコーディングする。
上記の表ではバックスラッシュ２つなっているが、ソースコード中では４つである。
