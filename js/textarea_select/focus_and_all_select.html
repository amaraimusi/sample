フォーカスしたときに値を全選択する

入力フォームをフォーカスしたとき、内部の文字列を全選択する。

サンプル

html

	<input id="test1" type="text" value="ネコがネズミを食べる" />
	<textarea  id="test2" cols="30" rows="10">ネコが&#13;魚用エサを盗み食いする。</textarea>

javascript/jquery

	$(function(){
		$("#test1").focus(function() {
		    $(this).select();
		});

		$("#test2").focus(function() {
		    $(this).select();
		});
	});
