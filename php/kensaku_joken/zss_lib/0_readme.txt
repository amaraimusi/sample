■zss_libについて

■使い方
開発するプロジェクトごとに、zss_libフォルダを丸ごとコピーし、開発プロジェクトフォルダの直下へ配置する。
そうしないと、kensaku_joken関連などフォルダでまとめているクラス群がパスの関係で使用できない。最初にアクセスしたファイルがカレントパスとなるため。

■編集の注意
zss_libファイルを修正した場合、当ファイルの内容も更新すること。
common.phpのgetVersionZssLibのバージョン番号も更新すること。
