GETとテキスト出力
GETからデータを取得し、テキスト出力する。
テキスト出力はAJAXへの戻値として使える。

aspxファイルの主要部分を抜粋

			
        protected void Page_Load(object sender, EventArgs e)
        {
            //GETパラメータを取得。
            String neko = Page.Request.QueryString.Get("neko");
            
            //テキスト出力。
            Page.Controls.Add(new LiteralControl("<p>ハローワールド</p>"));    
            Page.Controls.Add(new LiteralControl("<p>"+neko+"</p>"));
        }

GETパラメータ付のURL例

				http://localhost:3783/WebForm5.aspx?neko=12345

ブラウザでの出力結果


ハローワールド

12345
