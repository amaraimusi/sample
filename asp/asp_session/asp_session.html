ASP | SESSION
セッションから値を取り出す方法と、セッションへ値をセットする方法です。
セッション情報はブラウザに保存されるようなので、実行中のAspをいったん停止して再実行してもセッションの値は保存されたままになっています。
aspx.cs ソースコード 
			
protected void Button1_Click(object sender, EventArgs e)
{
    //セションから値を取り出す。
    string v=(string)Session["test_key"];
    Label1.Text = v;
}

protected void Button2_Click(object sender, EventArgs e)
{
    //セッションへ値を保存する。
    Session["test_key"] = TextBox2.Text;

}
